    1. Di sisi Frontend Development (Client-Side)


  • Menggunakan HTML untuk struktur halaman web dan CSS untuk styling. Mengerti prinsip desain responsif menggunakan media queries untuk memastikan tampilan website optimal di berbagai perangkat.
  • JavaScript (Vanilla JS): Mengembangkan interaktivitas web seperti validasi form, manipulasi DOM, animasi, dan AJAX tanpa menggunakan framework tambahan.
  • Framework JS: Berpengalaman menggunakan React atau Next.js untuk membuat aplikasi web dinamis dan komponen UI yang dapat digunakan kembali.
  • Responsive Design: Membuat desain web yang responsif dengan Bootstrap, Flexbox, atau CSS Grid untuk memastikan aplikasi tampil baik di berbagai ukuran layar.
  • Menggunakan Framework CSS seperti Tailwind CSS atau Bootstrap serta Framework Lainnya

    2. Di sisi Backend Development (Server-Side)


  • Node.js: Pengembangan aplikasi backend menggunakan JavaScript
  • Express.js: Framework web minimalis untuk Node.js yang memudahkan pembuatan API.
  • RESTful API: Desain dan implementasi API berbasis REST untuk komunikasi antara frontend dan backend.
  • MySQL: Database relasional untuk aplikasi dengan hubungan antar tabel.
  • Postman: Menguji dan mendokumentasikan API secara otomatis.
