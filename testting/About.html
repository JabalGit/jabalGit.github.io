About me

Nama saya Jabal nur , saya seorang Full Stack Web Developer, saya memiliki skill di bidang pengembangan web, baik di sisi front-end maupun back-end. Tujuan saya adalah menciptakan aplikasi web yang responsif, efisien, dan bermanfaat bagi pengguna dalam memakainya. Dengan pengalaman menggunakan HTML, CSS, JavaScript, serta framework seperti React dan Next.js untuk front-end, saya membangun antarmuka pengguna yang interaktif dan responsif. Di sisi back-end, saya bekerja dengan Node.js, Express.js, dan berbagai jenis database seperti Xampp dan MySQL untuk memastikan kelancaran operasional dan pengelolaan data di server. Saya senang bekerja dalam lingkungan maupun secara individu. jujur saja saya belum memiliki pengalaman kerja di industri, karena saya baru memasuki dunia pemrograman di 2024, umur saya masih 15 tahun dan saya masih sekolah di bangku SMA. itu telah menjadi tantangan bagi saya untuk terus berusaha mencapai impian, serta selalu berkomitmen untuk terus belajar dan berkembang.
